How to:
  1. Select a faction from the dropdown box.
  2. Enter the amount of heroes that you own in the fields.
  3. See what heroes can be fused. By clicking the fuse button you can fuse that hero
  4. See what heroes and fodder was used to fuse your hero of choice.
    Fused (in order):
    Valkyrie 7★

    Used copies:

Other usefull information:
  • Ent Elder : Heroes shown like this are considered trash and can not be upgraded to 6★. Use then as 5★ fodder.
  • Thale: Heroes shown like this are considered fodder.
  • Valkyrie: Heroes shown like this are natural 5★ heroes and in general should not be used as fodder.
  • Valkyrie: Heroes shown like this can potentially be upgraded to its listed ★ level. This is only an estimation and it is done based on the available 5★ heroes.

The game automaticly saves entered data. You can also save data that can be loaded at any time.

This tool allows you to enter all you heroes you own or have summoned. When you have enough to fuse any hero a fuse button will show. You can simulate that hero fusion to see what it will cost and what the results are.
